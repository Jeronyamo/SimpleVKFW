# CG Task 2

## Реализовано:
* Базовая часть (рендерер в *"source/renderer.h"*, дифференцируемый рендерер в *"source/differnderer.h"*);
* Edge sampling (треугольники, SDF круги);
* Из SDF убран smoothstep;
* Восстановление текстур высокого разрешения;

## Сборка:
Один исходник *"/drender.cpp"*, сборка через g++|cmake, сборка/запуск через *"/run.sh"*.

## Работа программы:
Три примера работы - Один из дополнительной части (восстановление *"./01_reference.png"*), а также два SDF изображения размером 256x256 и 64x64 из логотипа AC/DC (1024x1024). Промежуточные изображения (каждые 10 итераций) сохраняются в папки *"./res_triangle"*, *"./res_sdf"*, *"./res_sdf_smol"* соответственно, последнее изображение считается результатом работы.
